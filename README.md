# PyCacheSimulator
A parameterizable cache simulator
 
For running simulation, execute Code 1. Trace file is "trace.din", 16 is size of cache in KB, 32 is the size of block in byte, and 4 is the number of sets. For direct mapping the
size of set should be set to 1.

Code 1: How to execute the program.
$ ./CacheSimulator.py trace.din 16 32 4
